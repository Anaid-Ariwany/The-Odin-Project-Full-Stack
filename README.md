# The Odin Project - Full-Stack JavaScript

[Live Repo](https://anaid-ariwany.github.io/The-Odin-Project-Full-Stack/)

This repository contains my learning projects and exercises from The Odin Project's Full Stack JavaScript curriculum. It serves as a personal record of my progress and a showcase of the skills I've acquired along the way.

### Learning Path
This repository contains the fundamentals learning path, which includes projects and exercises focused on:

 - Intermediate HTML & CSS
 - JavaScript
 - Advanced HTML & CSS
 - React
 - Databases
 - Node.js

Each assignment and project is designed to reinforce the skills learned in the curriculum and provide practical experience in building web applications.

### Folder Structure
- Intermediate-HTML-CSS/: Contains intermediate HTML & CSS projects and exercises.
- Advanced-HTML-CSS/: Contains advanced HTML & CSS projects and exercises.
- JavaScript/: Contains intermediate and advanced JavaScript projects and exercises.
- React/: Contains React projects and exercises.
- Databases/: Contains projects and exercises related to databases.
- Node.js/: Contains Node.js projects and exercises.
- ./Projects/: Contains larger projects that integrate multiple skills.

### Technologies Used
- HTML5
- CSS3
- JavaScript (ES6+)
- React
- Node.js
- Express.js
- Database technologies (e.g., MongoDB, PostgreSQL)
- Git and GitHub for version control
