# Advanced HTML & CSS
This section of the curriculum is designed to build upon the intermediate knowledge of HTML and CSS, introducing more advanced concepts and techniques.

### Learning Path
This folder contains the fundamentals learning path, which includes projects and exercises focused on:

- Animation
- Accessibility
- Responsive Design

### Folder Structure
- Exercises/: Contains exercises that reinforce the concepts learned in the curriculum.
- Projects/: Contains larger projects that integrate multiple skills.
- Notes/: Contains notes and resources related to the curriculum.
