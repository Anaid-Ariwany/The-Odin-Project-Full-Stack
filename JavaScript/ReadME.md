# Intermediate & Advanced JavaScript
This section of the curriculum is designed to build upon the foundational knowledge of JavaScript, introducing more advanced concepts and techniques.

### Learning Path
This folder contains the fundamentals learning path, which includes projects and exercises focused on:

- Organizing JS Code
- JS in the Real World
- Asynchronous JS & APIs
- Testing JS
- A Bit of Computer Science
- Intermediate Git
- Final Project

### Folder Structure
- Exercises/: Contains exercises that reinforce the concepts learned in the curriculum.
- Projects/: Contains larger projects that integrate multiple skills.
- Notes/: Contains notes and resources related to the curriculum.
